# Flash-Chat iOS App

## 📌 Overview
Flash Chat is an internet-based messaging app similar to WhatsApp. It allows users to send and receive messages in real-time using Firebase Firestore as a backend database. This project focuses on building a chat application with an intuitive user interface and smooth user experience.

## 🎨 Features
✅ Real-time chat functionality using Firebase Firestore  
🔑 User authentication, registration, and login via Firebase  
📨 Send and receive messages instantly  
🎨 Custom UI with reusable cells  
📂 Persistent chat history  
🔍 User-friendly and minimalistic design  

## 🛠️ Tech Stack
- **Swift** – Primary programming language  
- **UIKit** – User interface development  
- **Firebase Firestore** – Cloud-based database  
- **CocoaPods & Swift Package Manager** – Dependency management  
- **Custom UITableViewCells** – Custom chat bubbles and UI components

## 🚀 Installation
Clone this repository:
```sh
git clone https://github.com/Nawazish2026/Flash-Chat-iOS.git
```
Navigate to the project folder:
```sh
cd Flash-Chat-iOS
```
Install dependencies using CocoaPods:
```sh
pod install
```
Open the `.xcworkspace` file in Xcode:
```sh
open FlashChat.xcworkspace
```
Run the project in an iOS simulator or a physical device.

## 🛠️ Dependencies
This project uses the following CocoaPods:
- **Firebase/Firestore** – Cloud database for storing and retrieving messages  
- **Firebase/Auth** – User authentication and login management
